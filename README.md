Budget Planner

A simple React-based tool for tracking monthly income, expenses, yearly costs, and savings. Everything updates automatically and is saved locally in your browser.

Features

Add income, bonuses, expenses, and yearly costs

Fixed vs flexible expenses

Automatic calculations (net, savings rate, totals)

Category breakdown and charts

Import/export JSON and CSV

Light/Dark mode

Setup
npm install
npm run dev


Open the app at:

http://localhost:5173

Build
npm run build
